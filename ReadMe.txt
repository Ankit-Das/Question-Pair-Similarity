Kind-Red Question-Pair-Similarity

#Objective:This project is built to identify and display similar questions that have been already answered on a forum on a web application.

#The dataset is webscrapped from a forum by using scrappy.

#Dataset is cleaned by removing tags,expanding english language and regex function.

#Word Embedding procedure(infersent) and cosine similarity are techniques used to find similar questions.

#Use argmax to compare the index of the question and dataset.It will give the most similar question the question asked. 

#Use argsort to compare the index of the question and dataset.Top similar questions will be displayed.

#Deployment is done using flask app.The result will be displayed on a web application
